This course teaches you to build websites from the ground up. You'll start with the fundamentals of how the web works, then learn HTML for structure, CSS for design, JavaScript for interactivity, and Python Flask for backend logic.

## What you'll learn

- How the internet and web servers work
- Semantic HTML that's accessible and well-structured
- CSS styling, layout, and responsive design
- JavaScript for interactive pages
- Python Flask for backend web development
- Building and deploying a complete portfolio site

## Prerequisites

Basic Python knowledge (variables, functions, lists, dictionaries) — covered in the "Getting Started with Python" course.
